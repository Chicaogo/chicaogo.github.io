种群及群落（2019年6月18日）

种群

同一时间，同一地点，同一生物。

种群特征

  • 最基本特征： 种群密度
  • 出生率、死亡率
  • 迁入率、迁出率
  • 性别组成、年龄组成（增长型、稳定型、衰退型）

空间结构

  • 均匀分布、集群分布、随机分布

调查种群密度的方法

样方法

  • 适用生物：植物、活动范围小/活动能力弱的动物

  • 取样：五点取样法、等距取样法

  • 计数方法：内部+取上不取下 or 内部+两邻边及夹角

标志重捕法

  • 适用生物：活动范围大、活动能力强
  • 标记物脱落，N偏大，实际值偏小
  • 第二次捕捉困难，N偏大

取样器取样法

  • 土壤小动物
  • 土壤小动物丰富度（生物种类）
  • 计数方法：
    • 记名计算法：大而少
    • 目测估计法：多程等级

现代生物进化理论(2019年6月19日)

现代生物进化理论的由来

拉马克

  • 不是神创造的，由更古老的生物进化而来
  • 由低等向高等进化
  • 生物的各种适应性特征的形成都是由于用尽废退和获得性遗传

达尔文：自然选择学说

  • 生物是不断进化的
  • 达尔文对遗传和变异的本质不做出解释
  • 对进化的解释局限于个体水平
  • 强调物种的形成都是渐变的结果，不能很好的解释物种大爆发

现代生物进化理论

  • 以生物个体为单位，发展到以种群为基本单位
  • 自然选择作用于生物个体的表现型
  • 概念区分
    • 种群
    • 物种：生产可育后代
    • 群落
    • 生态系统

进化的原材料——突变和基因重组

  • 可遗传变异是生物进化的原材料
  • 基因突变产生新的等位基因，So种群基因频率发生改变
  • 突变和重组都是随机的、不定向的，只能提供进化的原材料，不能决定生物进化的方向
  • 进化的方向：自然选择（环境）
  • 进化的本质：基因频率的定向改变

新物种的形成

  • 步骤：突变和重组 -> 自然选择 -> 隔离 -> 新物种

  • 隔离是物种形成的必要条件

  • 生殖隔离和地理隔离

生物多样性

  • 基因、物种、生态系统
  • 共同进化：不同物种之间、生物与无机环境之间在相互影响中不断进化和发展

呼吸作用

基本过程

有氧呼吸

细胞质基质

  • $C_6H_{12}O_6 → 2C_3H_4O_3(丙酮酸) + 4[H] + 2ATP + 热能 (少量能量)$

线粒体基质

  • $2C_3H_4O_3(丙酮酸) + 6H_2O → 6CO_2 + 20[H] + 2ATP + 热能​$

线粒体内膜

  • $24[H] + 6O_2 → 12H_2O + 大量能量$

总反应方程式

  • $C_6H_{12}O_6 + 6H_2O + 6O_2 → 6CO_2 + 12H_2O + 大量能量​$

无氧呼吸（细胞质基质中进行）

金鱼、植物、微生物、酵母菌

  • $C_6H_{12}O_6 → 2C_2H_5OH + 2CO_2 + 少量能量​$

动物肌肉、土豆块茎、乳酸菌

  • $C_6H_{12}O_6 → 2C_3H_6O_3(乳酸) + 少量能量$

人无氧呼吸不产生二氧化碳

呼吸熵&计算

$RQ = \frac{V_{CO_2}}{V_{O_2}}$

$C_6H_{12}O_6 → 2C_2H_5OH + 2CO_2 + 少量能量$

$C_6H_{12}O_6 + 6H_2O + 6O_2 → 6CO_2 + 12H_2O + 大量能量$

$RQ = 1$ 有氧呼吸

$RQ = 4/3$ 有氧 = 无氧

$RQ > 4/3$ 无氧 > 有氧
